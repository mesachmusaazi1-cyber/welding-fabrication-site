# Local Development Setup Guide\n\n## Prerequisites\n- Node.js v14+ and npm\n- Git\n- MongoDB (local or Atlas account)\n- Code editor (VS Code recommended)\n\n## Step 1: Clone the Repository\n\n```bash\ngit clone https://github.com/mesachmusaazi1-cyber/welding-fabrication-site.git\ncd welding-fabrication-site\n```\n\n## Step 2: Frontend Setup\n\n### Install Dependencies\n```bash\ncd frontend\nnpm install\n```\n\n### Create Environment File\n```bash\n# Create a .env file in the frontend directory\necho \"REACT_APP_API_URL=http://localhost:5000/api\" > .env\n```\n\n### Start Development Server\n```bash\nnpm start\n```\n\nThe frontend will open at `http://localhost:3000`\n\n## Step 3: Backend Setup\n\n### Install Dependencies\n```bash\ncd ../backend\nnpm install\n```\n\n### Configure Environment Variables\n```bash\ncp .env.example .env\n```\n\nEdit `.env` with your settings:\n```\n# Server\nPORT=5000\nNODE_ENV=development\n\n# Database - Use MongoDB Atlas or local MongoDB\nMONGODB_URI=mongodb://localhost:27017/welding-fabrication\n# OR for MongoDB Atlas:\n# MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/welding-fabrication\n\n# JWT\nJWT_SECRET=your-super-secret-key-change-this-in-production\nJWT_EXPIRE=7d\n\n# Email (Gmail)\nEMAIL_HOST=smtp.gmail.com\nEMAIL_PORT=587\nEMAIL_USER=your-email@gmail.com\nEMAIL_PASS=your-app-password\nEMAIL_FROM=noreply@weldingfab.com\n\n# Frontend\nFRONTEND_URL=http://localhost:3000\n```\n\n### Start Backend Server\n```bash\nnpm start\n# or for development with auto-reload:\nnpm run dev\n```\n\nThe backend API will run on `http://localhost:5000`\n\n## Step 4: Test the Application\n\n### Check Frontend\n- Navigate to `http://localhost:3000`\n- Test all pages:\n  - Home\n  - Services\n  - Portfolio\n  - Contact\n  - Quotation Request\n  - About\n  - Blog\n  - Admin Dashboard\n\n### Check Backend\n```bash\ncurl http://localhost:5000/api/health\n```\n\nShould return:\n```json\n{\"status\":\"API is running\"}\n```\n\n## Step 5: MongoDB Setup\n\n### Option A: Local MongoDB\n\n1. **Install MongoDB**\n   - Download from: https://www.mongodb.com/try/download/community\n   - Follow installation instructions\n\n2. **Start MongoDB**\n   ```bash\n   # On Windows\n   mongod\n   \n   # On Mac (with Homebrew)\n   brew services start mongodb-community\n   \n   # On Linux\n   sudo systemctl start mongod\n   ```\n\n3. **Update .env**\n   ```\n   MONGODB_URI=mongodb://localhost:27017/welding-fabrication\n   ```\n\n### Option B: MongoDB Atlas (Cloud)\n\n1. **Create Account**\n   - Go to: https://www.mongodb.com/cloud/atlas\n   - Sign up for free account\n\n2. **Create Cluster**\n   - Click \"Create a Deployment\"\n   - Choose \"Free\" tier (M0)\n   - Select region closest to you\n   - Create cluster\n\n3. **Create Database User**\n   - Go to \"Database Access\"\n   - Click \"Add New Database User\"\n   - Set username and password\n   - Add user\n\n4. **Get Connection String**\n   - Go to \"Databases\"\n   - Click \"Connect\"\n   - Choose \"Connect your application\"\n   - Copy connection string\n   - Replace `<password>` with your password\n\n5. **Update .env**\n   ```\n   MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/welding-fabrication\n   ```\n\n## Step 6: Email Configuration\n\n### Gmail Setup for Email Notifications\n\n1. **Enable 2-Factor Authentication**\n   - Go to: https://myaccount.google.com/security\n   - Enable 2-Step Verification\n\n2. **Generate App Password**\n   - Go to: https://myaccount.google.com/apppasswords\n   - Select \"Mail\" and \"Windows Computer\"\n   - Google will generate a 16-character password\n   - Copy this password\n\n3. **Update .env**\n   ```\n   EMAIL_USER=your-email@gmail.com\n   EMAIL_PASS=your-16-character-app-password\n   ```\n\n## Troubleshooting\n\n### Port Already in Use\n```bash\n# Find process using port 5000\nlsof -i :5000\n# Kill the process\nkill -9 <PID>\n```\n\n### MongoDB Connection Error\n- Ensure MongoDB is running\n- Check MONGODB_URI in .env\n- For Atlas: Check IP whitelist (allow all: 0.0.0.0/0)\n\n### npm install fails\n```bash\n# Clear npm cache\nnpm cache clean --force\n# Delete node_modules\nrm -rf node_modules\n# Reinstall\nnpm install\n```\n\n### Hot reload not working\n- Ensure NODE_ENV=development in .env\n- Restart both servers\n\n## Project Structure\n\n```\nwelding-fabrication-site/\nв”њв”Ђв”Ђ frontend/\nв”‚   в”њв”Ђв”Ђ public/\nв”‚   в”‚   в””в”Ђв”Ђ index.html\nв”‚   в”њв”Ђв”Ђ src/\nв”‚   в”‚   в”њв”Ђв”Ђ
